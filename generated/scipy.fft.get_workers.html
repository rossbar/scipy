scipy.fft.get_workers¶

scipy.fft.get_workers()[source]¶

Returns the default number of workers within the current context

Examples

>>> from scipy import fft
>>> fft.get_workers()
1
>>> with fft.set_workers(4):
...     fft.get_workers()
4
